# compile tests 

These shields provide several different kinds of ways to include the cirque on a keyboard.
I'm using them to check that compilation still works as expected.

# Shields

- **UsbOnly01**:
    - wired
    - i2c cirque
    - on central
    - usb only - ble disabled
- **UsbOnly02**:
    - wired
    - i2c cirque
    - on peripheral
    - usb only - ble disabled
- **Ble01**:
    - wireless
    - spi cirque
    - on central
    - ble only - usb disabled
- **Ble02**:
    - wireless
    - spi cirque
    - on peripheral
    - ble only - usb disabled
